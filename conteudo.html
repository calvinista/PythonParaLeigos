O que é Python?



Python é uma linguagem de programação open-source (código aberto), de propósito geral, interpretada e de alto nível. Sendo inicialmente criada na década de 1990 pelo matemático e programador holândes, Guido van Rossum, e bastante utilizada em aplicações de data science, machine learning, desenvolvimento web e de aplicativos, automação de processos e muito mais.



Aplicações



Por se tratar de uma linguagem de programação poderosa e versátil como Python, há um amplo leque de possíveis aplicações para ela. Tendo como principais aplicações:
  • Desenvolvimento Web;

  • Ciência de Dados;

  • Automatização de Tarefas por Scripting.
