APLICAÇÕES DA LINGUAGEM PYTHON


Ao começarmos a estudar uma nova linguagem de programação, nada melhor do que conhecer as suas aplicações, para então, decidir se esta atende nossas necessidades.

Com o Python não é diferente, e por se tratar de uma linguagem de programação poderosa e versátil, possui um amplo leque de aplicações possíveis. Tendo como principais aplicações:
  • DESENVOLVIMENTO WEB (DJANGO e FLASK): O desenvolvimento web, refere-se ao termo usado para definir sites ou aplicações que podem ser acessados diretamente por um browser, navegador web, seja em computadores ou dispositivos móveis. O Python oferece uma gama de frameworks, que são ferramentas essenciais para a construção do back end da web página, aquilo que está por trás de uma aplicação e não é visível ao usuário, sendo esses, os responsáveis pelas funções que ocorrem por trás de um site,como, atender à requisições entre cliente e servidor, lidar com o banco de dados e muitas outras funções que ocorrem por trás do site, e não são visíveis ao usuário.

    Abaixo há um artigo sobre os principais frameworks de desenvolvimento web em Python, creditada à página TREINAWEB_, falando mais detalhadamente sobre os dois principais frameworks de Python, Django e Flask. Sinta-se à vontade para desfrutar do conteúdo e saber um pouco dessas ferramentas que nos permitem fazer coisas tão maravilhosas!


  • CIÊNCIA DE DADOS: Segundo o Instituto Gartner, é possível que 2,2 bilhões de terabytes de novos dados sejam gerados diariamente. Com o intuito de analisar, armazenar, processar, vizualizar e executar várias manipulações com grandes volumes de dados, o Python tem bibliotecas com funções que, além de muito úteis, são essenciais na hora de lidar com os dados. Entre as bibliotecas mais usadas em Python temos: Scikit-learn, TensorFlow, Numpy, Pandas e SciPy, que além da ciência de dados, também são usadas em áreas como matemática, engenharia e análise de dados.


  • AUTOMATIZAÇÃO DE TAREFAS POR SCRIPTING: O termo Scripting, refere-se a escrita de pequenos programas projetados para a automatização de tarefas. Para esse tipo de trabalho, o Python torna-se a linguagem adequada, em razão de sua sintaxe relativamente simples e fácil de escrever, além da sua velocidade, já que é rápido escrever algo pequeno e testá-lo.
