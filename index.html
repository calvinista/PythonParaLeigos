Olá internautas! Tudo bem? Sejam bem vindos ao Python Para Leigos, um site desenvolvido como forma de trabalho avaliativo por estudante de Programação Web FullStack, cursista de um programa financiado pela Dell Technologies, e ofertado pelo Instituto da Oportunidade Social (IOS).



Aqui, serão abordados conhecimentos para os interessados nas capacidades dessa linguagem de programação, que está crescendo cada vez mais, o Python. Conhecimentos esses que vão desde noções básicas sobre Python até suas as inúmeras aplicações, comandos, história e curiosidades.

Espera um pouquinho! então significa que você esta começando do zero e não tem nenhuma noção sobre Python? Não se preocupe! Apenas navegue e desfrute de todo o conteúdo e conhecimento desse humilde site feito com carinho de um leigo para o outro.
